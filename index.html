About the Dataset

Dataset Overview

The dataset used in this project is entirely created by me from scratch, with AI assistance, to simulate a realistic retail and e-commerce business scenario. It is not publically available anywhere. It includes multiple interconnected tables covering different aspects of operations.

  • Type: Simulated/Mock business dataset.
  • Source: Self-created in Excel, imported into SQL Server.
  • Data Volume: ~15000 total rows across all tables (8,000+ in OrderDetails alone).
  • Time Period Covered: Q1 (January - March 2025). 3 month simulated dataset.
  • Domain: Business Operations â€“ Consumer Electronics.

Tables

The database consists of six core tables, each serving a unique function:

  • Orders â€“ Records each transaction along with order date, channel, and delivery status.
  • OrderDetails â€“ Links orders and products, capturing quantity, sales price, and item-level data.
  • Products â€“ Contains details of all items available for sale, including price, cost, and launch date.
  • Customers â€“ Stores customer information such as names, location, age group, etc.
  • Returns â€“ Tracks returned items, reasons, and related order links.
  • Reviews â€“ Includes customer feedback, ratings, and review dates for purchased products.

Query Showcase

Basic Queries

Question: How many unique customers do we have in total?

SQL 

SELECT DISTINCT count(customerId) AS 'No. of Unique Customers' 
FROM Customers;

Question: What is the gender distribution of our customers?

SQL 

SELECT Gender,
       Count(Customerid) AS 'No. of Customers'
FROM customers
GROUP BY Gender
ORDER BY 'No. of Customers' DESC;

Question: What is the average age of our customers?

SQL 

SELECT round(Avg(age), 0) AS 'Average Age of Customers'
FROM customers

Question: How many unique products are listed in each subcategory?

SQL 

SELECT subcategory,
       count(Productid) AS 'No. of Products'
FROM products
GROUP BY Subcategory
ORDER BY 'No. of Products' DESC;

Question: Which product has the highest MRP?

SQL 

SELECT Top 1 ProductId,
           ProductName,
           MRP
FROM Products
ORDER BY MRP DESC

Question: Which subcategory has the highest average MRP?

SQL 

SELECT Top 1 SubCategory,
           round(Avg(MRP), 2) AS 'Average MRP'
FROM Products
GROUP BY Subcategory
ORDER BY 'Average MRP' DESC;

Question: What is our total revenue for Jan, Feb, and March?

SQL 

SELECT Format(Orderdate, 'MMMM') AS 'Months',
       Sum(TotalAmount) AS 'Total Revenue'
FROM orders
GROUP BY Format(Orderdate, 'MMMM'),
         Month(OrderDate)
ORDER BY month(orderdate)

Question: How many orders were placed via each sales channel?

SQL 

SELECT OrderChannel,
       Count(orderid) AS 'No. of orders'
FROM orders
GROUP BY orderchannel
ORDER BY 'No. of orders' DESC;

Question: What is the average order value per channel?

SQL 

SELECT OrderChannel,
       Round(Avg(TotalAmount), 2) AS 'Avg order value'
FROM orders
GROUP BY Orderchanne;

Question: What are the 3 most common return reasons?

SQL 

SELECT Top 3 Reason,
           Count(returnid) 'No of Returns'
FROM RETURNS
GROUP BY Reason
ORDER BY 'No of Returns' DESC

Intermediate Queries

Question: Which states have the highest number of registered customers?

SQL 

SELECT Top 1 State,
           count(customerid) AS 'No. of customer'
FROM customers
GROUP BY state
ORDER BY 'No. of customer' DESC

Question: How many customers registered in each month?

SQL 

SELECT Format(RegistrationDate, 'MMM/yyyy') AS 'Registration Month',
       Count(customerid) AS 'No. of registrations'
FROM customers
GROUP BY Format(RegistrationDate, 'MMM/yyyy'),
         year(RegistrationDate),
         month(RegistrationDate)
ORDER BY year(RegistrationDate),
         month(RegistrationDate)

Question: Which product has sold the most units overall?

SQL 

SELECT Top 1 Orderdetails.ProductID,
           Productname,
           Sum(Quantity) AS 'Total Quantity Sold'
FROM OrderDetails
INNER JOIN products ON OrderDetails.ProductID=Products.ProductId
GROUP BY orderdetails.ProductID,
         Products.ProductName
ORDER BY 'Total Quantity Sold' DESC

Question: Which subcategories have generated the highest sales revenue?

SQL 

SELECT SubCategory,
       Sum(SalesPrice) AS 'Revenue'
FROM OrderDetails
INNER JOIN Products ON Orderdetails.ProductID=products.ProductId
GROUP BY SubCategory
ORDER BY 'Revenue' DESC

Question: How much discount did we give overall and per month?

SQL 

SELECT Format(Orderdate, 'MMM-yyyy') AS 'Order Month',
       Sum((Quantity*UnitPrice)-SalesPrice) AS 'Discount Amount'
FROM OrderDetails
INNER JOIN orders ON OrderDetails.OrderID=Orders.OrderID
GROUP BY Format(Orderdate, 'MMM-yyyy'),
         month(orderdate)
ORDER BY month(orderdate) 

--And--

SELECT Sum((Quantity*UnitPrice)-SalesPrice) AS 'Total Discount Amount'
FROM OrderDetails

Question: What is the total revenue generated by each subcategory?

SQL 

SELECT SubCategory,
       sum(SalesPrice) AS 'Total Revenue'
FROM OrderDetails
INNER JOIN Products ON OrderDetails.ProductID=Products.ProductId
GROUP BY SubCategory
ORDER BY [Total Revenue] DESC

Question: Which 5 cities brought the highest revenue?

SQL 

SELECT Top 5 ShippingCity,
           Sum(Totalamount) AS 'Total Revenue'
FROM orders
GROUP BY ShippingCity
ORDER BY [Total Revenue] DESC

Question: Which are the top 3 products with the highest return counts?

SQL 

SELECT Top 3 Products.ProductId,
           products.ProductName,
           count(returns.returnid) AS 'No of returns'
FROM RETURNS
INNER JOIN orderdetails ON returns.orderID=OrderDetails.OrderID
INNER JOIN Products ON orderdetails.ProductID=products.ProductId
GROUP BY products.ProductId,
         products.productName
ORDER BY 'No of returns' DESC

Question: What is the average delivery time ?

SQL 

SELECT concat(Avg(datediff(DAY, orderdate, Deliverydate)), ' Days') 
AS 'Average Delivery Time'
FROM orders

Question: Which products have the most positive reviews?

SQL 

SELECT Top 3 reviews.ProductID,
           products.ProductName,
           Count(*) AS 'No. of Positive Reviews'
FROM Reviews
INNER JOIN Products ON reviews.ProductID=products.ProductId
WHERE ReviewTag IN('Excellent',
                   'Good Quality',
                   'Loved it',
                   'Highly Recommend')
GROUP BY reviews.ProductID,
         products.productname
ORDER BY 'No. of Positive Reviews' DESC

Advanced Queries

Question: Which customer age group contributes to the most orders?

SQL 

WITH AgeGroups AS
  (SELECT Customerid,
          CASE
              WHEN age<=25 THEN '16-25'
              WHEN age BETWEEN 26 AND 35 THEN '26-35'
              WHEN age BETWEEN 36 AND 45 THEN '36-45'
              WHEN age BETWEEN 46 AND 55 THEN '46-55'
              ELSE '55+'
          END AS Age
   FROM customers)
SELECT agegroups.Age,
       Count(orders.CustomerId) AS 'No. of orders'
FROM AgeGroups
INNER JOIN orders ON Agegroups.CustomerId=orders.CustomerId
GROUP BY AgeGroups.Age
ORDER BY [No. of orders] DESC

Question: Which state and age group combination brings the highest revenue?

SQL 

WITH AgeGroups AS
  (SELECT Customerid,
          CASE
              WHEN age <= 25 THEN '16-25'
              WHEN age BETWEEN 26 AND 35 THEN '25-35'
              WHEN age BETWEEN 36 AND 45 THEN '35-45'
              WHEN age BETWEEN 46 AND 55 THEN '45-55'
              ELSE '55+'
          END AS Age
   FROM customers)
SELECT top 3 ShippingState,
           Age,
           sum(totalamount) AS 'Total Revenue'
FROM orders
INNER JOIN AgeGroups ON orders.CustomerId=AgeGroups.CustomerId
GROUP BY ShippingState,
         Age
ORDER BY [Total Revenue] DESC

Question: Which are the Top 3 newly launched products (in the last 30 days) with the highest total sales revenue?

SQL 

 SELECT Top 3 products.Productid,
           products.ProductName,
           LaunchDate,
           sum(Orderdetails.SalesPrice) AS 'Revenue'
FROM OrderDetails
RIGHT JOIN products ON OrderDetails.ProductID=products.ProductId
WHERE LaunchDate>='2025-3-1'
GROUP BY Products.ProductId,
         products.ProductName,
         LaunchDate
ORDER BY Revenue DESC

Question: What is our gross profit and profit margin overall and per month?

SQL 

SELECT sum(salesprice) AS 'TotalRevenue',
       sum(costprice*quantity) AS 'Total cost',
       (sum(salesprice)-sum(costprice*quantity)) AS 'Overall Gross Profit',
       concat(Round(((sum(salesprice)-sum(costprice*quantity))*100)/nullif(sum(salesprice), 0), 2), '%') AS 'Profit Margin'
FROM orderdetails
INNER JOIN products ON orderdetails.productid=products.productid 

---AND---

SELECT format(orders.orderdate, 'MMMM-yyyy') AS 'Months',
       sum(salesprice) AS 'TotalRevenue',
       sum(costprice*quantity) AS 'Total cost',
       (sum(salesprice)-sum(costprice*quantity)) AS 'Gross Profit',
       concat(Round(((sum(salesprice)-sum(costprice*quantity))*100)/nullif(sum(salesprice), 0), 2), '%') AS 'Profit Margin'
FROM orderdetails
INNER JOIN products ON orderdetails.productid=products.productid
INNER JOIN Orders ON OrderDetails.OrderID=Orders.OrderID
GROUP BY format(orders.orderdate, 'MMMM-yyyy'),
         month(orderdate)
ORDER BY month(orderdate)

Question: Which channels (App/Website/Flipkart) generate higher profit margins?

SQL 

SELECT OrderChannel,
       Format(((sum(salesprice)-sum(costprice*quantity)))/nullif(sum(salesprice),0),'P2') as 'Profit Margin'
FROM orderdetails
INNER JOIN products ON orderdetails.productid=products.productid
INNER JOIN Orders ON OrderDetails.OrderID=Orders.OrderID
GROUP BY OrderChannel

Question: What is the return rate of each product compared to its total number of orders?

SQL 

SELECT products.ProductName,
       COUNT(DISTINCT orderdetails.orderid) AS [Total Orders],
       COUNT(DISTINCT returns.ReturnID) AS [Total Returns],
       concat(COUNT(DISTINCT returns.ReturnID)*100/NULLIF(COUNT(DISTINCT orderdetails.OrderID),0),'%') AS 'Return Rate (%)'
FROM orderdetails
INNER JOIN Products ON orderdetails.ProductID = products.ProductID
LEFT JOIN RETURNS ON orderdetails.OrderID = returns.OrderID
GROUP BY products.ProductName
ORDER BY COUNT(DISTINCT returns.ReturnID)*100/NULLIF(COUNT(DISTINCT orderdetails.OrderID),0) DESC;

-- NOTE: Used ORDER BY COUNT(DISTINCT returns.ReturnID)*100/NULLIF(COUNT(DISTINCT orderdetails.OrderID),0) DESC
-- instead of ORDER BY [Return Rate (%)] ; because CONCAT converts numeric values to text, 
-- which prevents correct numerical sorting.
